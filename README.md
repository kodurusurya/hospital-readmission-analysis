# Hospital Readmission Analysis

## Project Overview
This project analyzes hospital encounter data for diabetic patients to identify factors associated with 30-day readmissions.

## Business Problem
Hospital readmissions increase healthcare costs, reduce bed availability, and may indicate gaps in discharge planning and care coordination.

## Tools Used
- SQL
- Python
- Tableau
- Git
- GitHub

## Dataset
UCI Diabetes 130-US Hospitals Dataset

## Project Goals
- Measure 30-day readmission rate
- Identify high-risk patient groups
- Analyze clinical and hospital utilization factors
- Build a Tableau dashboard
- Provide business recommendations
